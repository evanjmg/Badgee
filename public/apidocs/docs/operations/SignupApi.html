signupPost

Sign up to Badgee

ENDPOINT

https://www.badgee.uk/api/signup

HTTP Method

POST

Response Type

See response object

Parameters

    Send the following in the body:
  • name

    provide full name

  • email

    must be a valid email address

  • password

    must be at least 8 characters

  • imgUrl

    provide image url for profile - make sure it's square
