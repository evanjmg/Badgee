tasksGet

Get a list of all the public challenges

ENDPOINT

https://www.badgee.uk/api/tasks

HTTP Method

GET

Response Type

Tasks

Parameters

tasksPost

Create challenges

ENDPOINT

https://www.badgee.uk/api/tasks

HTTP Method

POST

Response Type

Tasks

Parameters

    Send the following in the body:
  • lat String

    Send latitude of current user

  • lon String

    Send longitude of current user

  • task

    Send task object - see example:

    { "task":
    { "description": "d",
    "minutes": 9,
    "location.name": "name",
    "_creator": "560067f771cbc90300d58c38",
    "_tagged_member": "55ff3d3f523c55030003ec93"
    },
    "lat": "51.374277" ,
    "lon": "-0.101937" }
