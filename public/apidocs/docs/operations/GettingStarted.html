Getting Started

  1. Get Your API Key by making a post request to api/signup - click here to see the required parameters.
  2. Add the token you received from api/signup to your header like so: Authorization: Bearer TOKEN
  3. Start making requests to the other endpoints with that same header. There's no need to login again or get a new token.
  4. Make sure to keep your API KEY safe and put it in an environment variable
